# PDF Processing

Use the harness Read tool to inspect a PDF. Always invoke the harness Bash
tool for extraction, then ask the user to activate `/pdf-processing`
again before filling a form.

For form details, read `references/forms.md`.

Save output beside the source file. Assume Python and `pdftotext` are
already installed.
